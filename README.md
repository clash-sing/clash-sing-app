> 一款基于 Flutter 开发、集成 [sing-box](https://github.com/SagerNet/sing-box) 内核的跨平台代理客户端，支持 Clash 配置，助你轻松实现安全、高效的网络代理。

---

## 🌟 特性

- ✅ **跨平台支持**：一次开发，多端运行（Android / iOS / Windows / macOS / Linux）
- ⚡ **高性能代理**：底层使用 [sing-box](https://github.com/SagerNet/sing-box) 引擎，支持多种协议（如 VLESS、Trojan、Shadowsocks、HTTP(S)、SOCKS 等）
- 📦 **Clash 兼容**：原生支持 Clash Meta / Clash Premium 配置文件（`.yaml`）
- 🔒 **本地运行**：所有代理逻辑在设备本地执行，保障隐私与安全
- 🎨 **现代化 UI**：简洁直观的用户界面，操作流畅
- 🛠️ **灵活配置**：支持订阅链接、本地配置导入、规则编辑等高级功能
